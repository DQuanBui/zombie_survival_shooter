# Dead Zone: Zombie Survival Shooter

![](https://github.com/DQuanBui/zombie_survival_shooter/blob/main/zombie.jpg)

## Project Overview

**Dead Zone: Zombie Survival Shooter** is an interactive browser-based action game built with **HTML, CSS, and JavaScript**. The game challenges players to survive waves of zombies by moving, aiming, shooting, reloading, switching weapons, and managing health and stamina.

The project focuses on responsive gameplay, real-time rendering, organized front-end architecture, and an engaging user experience directly in the browser.

## Tools

This project uses:

- **HTML5** for the page structure and canvas setup
- **CSS3** for menus, HUD, layout, and visual styling
- **JavaScript** for gameplay logic and game systems
- **HTML5 Canvas API** for real-time rendering
- **JavaScript Modules** for organized code structure
- **VS Code** for development
- **Git and GitHub** for version control and project hosting
- **GitHub Pages** for online deployment

## Features

The game currently includes:

- Full-screen browser gameplay
- Start, pause, and game-over screens
- Player movement with mouse aiming
- Shooting and reload mechanics
- Weapon switching system
- Zombie wave progression
- Multiple zombie types
- Health, stamina, XP, score, and kill tracking
- Dash ability
- Screen shake and particle effects

## Gameplay

The player begins a survival mission and must fight against waves of zombies. Zombies spawn around the map and chase the player. The player must move strategically, aim accurately, reload at the right time, and use dash to escape dangerous situations. As the game progresses, waves become more challenging and the player must survive as long as possible. The game ends when the player's health reaches zero.

### Controls

| Action | Control |
|---|---|
| Move | W, A, S, D |
| Aim | Mouse |
| Shoot | Left Mouse Button |
| Reload | R |
| Dash | Shift |
| Switch Weapon | 1, 2, 3 |
| Pause | P |
